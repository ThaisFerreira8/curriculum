
# Curriculum

A modern full-stack portfolio and resume application built with Next.js, TypeScript, and Tailwind CSS.

The project was created to present my professional journey, software engineering experience, technical skills, and personal projects in a clean and interactive way.

## Goals

- Showcase professional experience
- Demonstrate frontend and backend development skills
- Apply modern web development practices
- Serve as a personal portfolio website

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- React
